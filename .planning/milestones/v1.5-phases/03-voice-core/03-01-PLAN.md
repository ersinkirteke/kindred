<task type="auto">
  <name>Task 2: ElevenLabs REST API service and R2 voice sample storage</name>
  <files>backend/src/voice/elevenlabs.service.ts, backend/src/images/r2-storage.service.ts</files>
  <action>
1. Create backend/src/voice/elevenlabs.service.ts:
   - @Injectable() NestJS service
   - Read ELEVENLABS_API_KEY from process.env (graceful initialization: log warning if missing, don't crash — matches Mapbox/Firebase pattern from Phase 1/2)
   - baseUrl = 'https://api.elevenlabs.io/v1'

   Methods:
   a) cloneVoice(params: { name: string; files: Buffer[] }): Promise<string>
      - POST to /voices/add with FormData (multipart/form-data)
      - Append each buffer as file with audio/mpeg mimetype
      - Return voice_id from response
      - Throw descriptive error on failure (include status code and ElevenLabs error message)

   b) deleteVoice(voiceId: string): Promise<void>
      - DELETE to /voices/{voiceId}
      - Swallow 404 errors (voice already deleted) — prevents Pitfall 5 from research

   c) generateSpeechStream(voiceId: string, text: string): Promise<ReadableStream>
      - POST to /text-to-speech/{voiceId}/stream
      - Use model_id: 'eleven_flash_v2_5' (ultra-low latency ~75ms per research)
      - voice_settings: { stability: 0.5, similarity_boost: 0.75 }
      - Return response.body (ReadableStream) for chunked streaming to client
      - Throw descriptive error on failure

   d) getVoice(voiceId: string): Promise<{ voice_id: string; name: string } | null>
      - GET to /voices/{voiceId}
      - Return voice data or null if 404 (for status checking during cloning)

2. Extend backend/src/images/r2-storage.service.ts with voice methods:
   a) uploadVoiceSample(userId: string, audioBuffer: Buffer): Promise<string>
      - Key: `voice-samples/${userId}/${Date.now()}.mp3`
      - ContentType: 'audio/mpeg'
      - Return public URL

   b) deleteVoiceSample(url: string): Promise<void>
      - Extract key from URL, delete from R2
      - Swallow errors if file already deleted

3. Add ElevenLabsService to VoiceModule providers. Add R2StorageService to VoiceModule imports (from ImagesModule).

4. Add ELEVENLABS_API_KEY to backend/.env.example with placeholder value.
