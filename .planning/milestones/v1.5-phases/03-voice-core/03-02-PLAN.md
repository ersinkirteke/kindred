  <action>
1. Create backend/src/voice/voice-cloning.processor.ts:
   - Reuse the in-memory queue pattern from ImageGenerationProcessor (Phase 1)
   - @Injectable() service with queue array and processing flag
   - enqueue(userId: string, voiceProfileId: string, audioUrl: string): void
     - Push to queue, update VoiceProfile status to PROCESSING
     - If not already processing, call processQueue()
   - processQueue(): async loop through queue items
     - Download audio buffer from R2 URL (fetch the audioSampleUrl)
     - Call elevenLabsService.cloneVoice({ name: `kindred_${voiceProfileId}`, files: [audioBuffer] })
     - On success: update VoiceProfile status=READY, elevenLabsVoiceId=returned_voice_id
     - Send push notification via pushService.sendToUser(userId, { title: "Your voice is ready!", body: "Start listening to recipes in your loved one's voice", data: { type: 'VOICE_READY', voiceProfileId } })
     - On failure: update VoiceProfile status=FAILED, log error
     - Process next item with setTimeout(100ms)
   - Inject: ElevenLabsService, PrismaService, PushService

2. Create backend/src/voice/voice.service.ts:
   - @Injectable() service
   - Inject: PrismaService, R2StorageService, VoiceCloningProcessor, ElevenLabsService

   Methods:
   a) uploadVoice(userId: string, audioBuffer: Buffer, input: UploadVoiceInput, ipAddress: string): Promise<VoiceProfile>
      - Validate consentGiven === true, throw BadRequestException if false ("Voice consent is required")
      - Check tier limits: query user's active voice profiles (status NOT IN ['DELETED', 'FAILED'])
      - If user.tier === 'FREE' (default to FREE if no tier field yet) and active count >= 1:
        throw ForbiddenException({ code: 'VOICE_SLOT_LIMIT', message: 'Free tier allows 1 voice. Upgrade to Pro for unlimited voices.', currentVoiceId: activeProfiles[0].id })
      - Upload audio to R2 via r2StorageService.uploadVoiceSample(userId, audioBuffer)
      - Create VoiceProfile record with status=PENDING, audioSampleUrl, speakerName, relationship, consentedAt=new Date(), consentIpAddress=ipAddress
      - Enqueue background cloning via voiceCloningProcessor.enqueue()
      - Return created VoiceProfile

   b) getVoiceProfiles(userId: string): Promise<VoiceProfile[]>
      - Return all non-DELETED voice profiles for user, ordered by createdAt desc

   c) getVoiceProfile(id: string, userId: string): Promise<VoiceProfile>
      - Find by id and userId (ownership check), throw NotFoundException if not found

   d) deleteVoiceProfile(id: string, userId: string): Promise<VoiceProfile>
      - Find profile, verify ownership
      - If elevenLabsVoiceId exists: call elevenLabsService.deleteVoice() (swallow errors)
      - Delete R2 voice sample via r2StorageService.deleteVoiceSample()
      - Update status to DELETED
      - Return updated profile

   e) replaceVoice(id: string, userId: string, audioBuffer: Buffer, ipAddress: string): Promise<VoiceProfile>
      - Find existing profile, verify ownership
      - Delete old ElevenLabs voice if exists
      - Delete old R2 file
      - Upload new audio to R2
      - Update profile: new audioSampleUrl, status=PENDING, consentedAt=now, consentIpAddress
      - Enqueue new cloning job
      - Return updated profile (implements VOICE-07 re-record)

3. Create backend/src/voice/voice.controller.ts:

   - @Controller('voice') with @UseGuards(ClerkAuthGuard)

     - Extract IP from request (req.ip or x-forwarded-for)
     - Call voiceService.uploadVoice()
   - POST /voice/:id/replace:
     - Same file upload pattern but calls voiceService.replaceVoice()
     - For VOICE-07 re-record flow
