  <files>

  <action>
    Create background audio infrastructure and lock screen control integration:

    **AudioSessionConfigurator.swift:**
    - Static method `configure()`:
      - `AVAudioSession.sharedInstance().setCategory(.playback, mode: .spokenAudio, options: [])` — .playback for background, .spokenAudio for voice narration (Claude's discretion per CONTEXT.md)
      - `session.setActive(true)`
      - Register for `.interruptionNotification` (phone calls, Siri)
      - Interruption handler: on `.began` → no action needed (AVPlayer auto-pauses), on `.ended` → check `.shouldResume` option before auto-resuming
      - Register for `.routeChangeNotification` (headphone unplug) — pause on old device disconnected to prevent unexpected speaker blaring
    - Call from AppDelegate.didFinishLaunching

    **NowPlayingManager.swift:**
    - Class with MPRemoteCommandCenter and MPNowPlayingInfoCenter integration
    - `setupRemoteCommands(onPlay:onPause:onSkipForward:onSkipBackward:onSeek:)`:
      - `commandCenter.playCommand` → calls onPlay
      - `commandCenter.pauseCommand` → calls onPause
      - `commandCenter.skipForwardCommand.preferredIntervals = [30]` → calls onSkipForward(30) — 30s per user decision
      - `commandCenter.skipBackwardCommand.preferredIntervals = [15]` → calls onSkipBackward(15) — 15s per user decision
      - `commandCenter.changePlaybackPositionCommand` → calls onSeek(positionTime) for lock screen seek bar
    - `updateNowPlaying(title:artist:artwork:duration:elapsedTime:rate:)`:
      - Set `MPMediaItemPropertyTitle` = recipe name
      - Set `MPMediaItemPropertyArtist` = speaker name (VOICE-03: speaker name prominent on lock screen)
      - Set `MPMediaItemPropertyPlaybackDuration` = duration
      - Set `MPNowPlayingInfoPropertyElapsedPlaybackTime` = elapsedTime
      - Set `MPNowPlayingInfoPropertyPlaybackRate` = rate
      - If artwork UIImage provided: create NEW `MPMediaItemArtwork` instance every time (per Research Pitfall 6 — must recreate, not reuse)
      - Use Kingfisher to download artwork image from URL before setting
    - `cleanup()`:
      - Remove all command targets with `.removeTarget(nil)`
      - Set `infoCenter.nowPlayingInfo = nil`
    - Make this a TCA-compatible dependency or call it directly from VoicePlaybackReducer effects

    **Info.plist modification:**
    - Add `UIBackgroundModes` key with array containing `audio` string
    - CRITICAL: Without this, background audio STOPS when screen locks (Research Pitfall 2)
    - Check if Info.plist exists at Kindred/Resources/Info.plist. If it exists, add the key. If it doesn't exist and background modes are configured via Xcode project settings or project.yml, add it to project.yml instead.
    - Also check Kindred/project.yml for existing configuration.
  </action>
