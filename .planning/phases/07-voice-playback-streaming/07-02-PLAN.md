  <files>Kindred/Packages/VoicePlaybackFeature/Sources/Player/VoicePlaybackReducer.swift</files>
  <action>
    Create the TCA reducer that manages all voice playback state:

    **State (@ObservableState, public):**
    - `currentPlayback: CurrentPlayback?` — nil when no active playback
    - `isExpanded: Bool = false` — mini-player expanded to bottom sheet
    - `voiceProfiles: [VoiceProfile] = []` — available voice profiles
    - `selectedVoiceId: String?` — currently selected voice for playback
    - `lastUsedVoicePerRecipe: [String: String] = [:]` — recipeId -> voiceId mapping stored in @AppStorage
    - `isLoadingNarration: Bool = false` — loading state for narration generation/fetch
    - `showVoicePicker: Bool = false` — voice picker visibility
    - `narrationMetadata: NarrationMetadata?` — step timestamps for current playback
    - `recipeSteps: [String] = []` — step text for display in expanded player
    - `error: String?`

    **Actions (public enum):**
    - `startPlayback(recipeId: String, recipeName: String, artworkURL: String?, steps: [String])` — initiate playback for a recipe
    - `voiceProfilesLoaded(Result<[VoiceProfile], Error>)` — voice profiles fetched
    - `selectVoice(String)` — user selects a voice from picker
    - `narrationReady(NarrationMetadata)` — narration audio URL and metadata received
    - `narrationFailed(String)` — narration generation/fetch failed
    - `play` — resume playback
    - `pause` — pause playback
    - `seekTo(TimeInterval)` — seek to specific time
    - `skipForward` — skip 30 seconds forward
    - `skipBackward` — skip 15 seconds backward
    - `cycleSpeed` — cycle to next PlaybackSpeed
    - `toggleExpanded` — toggle mini-player expansion
    - `dismiss` — stop playback and dismiss player
    - `timeUpdated(TimeInterval)` — from time observer stream
    - `durationUpdated(TimeInterval)` — from duration stream
    - `statusChanged(PlaybackStatus)` — from status stream
    - `switchVoiceMidPlayback(String)` — pause, fetch new voice, resume from same position (per user decision)
    - `previewVoiceSample(String)` — play 3-5 second voice sample in picker
    - `stopPreview` — stop voice sample preview
    - `cachingCompleted(URL)` — auto-cache saved successfully
    - `cachingFailed(String)` — auto-cache failed (non-critical)

    **Dependencies:**
    - `@Dependency(\.audioPlayerClient) var audioPlayer`
    - `@Dependency(\.voiceCacheClient) var voiceCache`
    - `@Dependency(\.continuousClock) var clock` (for delayed dismissal)

    **Key reducer logic:**

    1. **startPlayback:** Set isLoadingNarration = true, check lastUsedVoicePerRecipe for this recipe. If voice exists, skip picker (auto-start). If no voice, show picker (showVoicePicker = true). Fetch voice profiles. Store recipeSteps for expanded player display.

    2. **selectVoice:** Set selectedVoiceId, hide picker, set isLoadingNarration = true. Store in lastUsedVoicePerRecipe. Check voiceCache for existing audio. If cached: load from cache URL. If not cached: construct narration API URL (GraphQL mutation or R2 presigned URL), stream from remote. Create CurrentPlayback with initial values.

    3. **narrationReady:** Store metadata, create CurrentPlayback, start observing streams (time, status, duration) via `.run` effects with `.cancel(id:)` for lifecycle management. Call audioPlayer.play(url).

    4. **timeUpdated:** Update currentPlayback.currentTime. Call StepSyncEngine.currentStepIndex(at:timestamps:) to update currentStepIndex.

    5. **statusChanged:** Update currentPlayback.status. On `.stopped`: auto-dismiss after 2-second pause (mini-player fades per user decision). On `.playing`: trigger auto-cache if not already cached — download data from URL and call voiceCache.cacheAudio.

    6. **seekTo / skipForward / skipBackward:** Calculate target time (clamp to 0...duration). Call audioPlayer.seek. Update currentPlayback.currentTime immediately for responsive UI.

    7. **cycleSpeed:** Get next speed from currentPlayback.speed.next. Call audioPlayer.setRate. Update currentPlayback.speed.

    8. **switchVoiceMidPlayback:** Save currentTime. Call audioPlayer.pause(). Set isLoadingNarration = true (shows spinner on play button per user decision). Fetch new voice narration. On ready: audioPlayer.play(newURL), audioPlayer.seek(to: savedTime).

    9. **dismiss:** Call audioPlayer.cleanup(). Cancel all stream observation effects. Set currentPlayback to nil.

    Use `.cancellable(id:)` with enum CancelID for stream observations (timeObserver, statusObserver, durationObserver, autoCache, delayedDismiss).

    Make the reducer public with `public init() {}`.
  </action>
