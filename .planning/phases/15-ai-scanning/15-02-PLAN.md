**2. Create ScanResultsReducer.swift in Scanning/:**

@Reducer struct ScanResultsReducer with @ObservableState State:

State:
- scanType: ScanType
- photoUrl: String? (shown as header for fridge scans)
- detectedItems: IdentifiedArrayOf<DetectedItemState>
- isAdding: Bool = false
- addSuccess: Bool = false
- addedCount: Int = 0
- errorMessage: String?
- isAddingNewItem: Bool = false (for "+ Add item AI missed")
- newItemName: String = ""

DetectedItemState (nested struct, Identifiable, Equatable):
- id: UUID
- name: String
- quantity: String
- category: FoodCategory
- storageLocation: StorageLocation
- estimatedExpiryDays: Int
- confidence: Int
- isChecked: Bool (auto-set: confidence >= 70 -> true, else false)
- isEditing: Bool = false

Computed property `badgeColor`:
- confidence >= 90 -> Color.green
- confidence >= 70 -> Color.yellow
- else -> Color.red

Computed `checkedCount`: items.filter(\.isChecked).count

Init from DetectedItem: map all fields, auto-check based on confidence >= 70

Actions (BindableAction):
- binding(BindingAction<State>)
- itemToggled(id: UUID) -- toggle isChecked
- itemEditTapped(id: UUID) -- toggle isEditing (double-tap per user decision)
- itemFieldChanged(id: UUID, name: String?, quantity: String?, category: FoodCategory?, storageLocation: StorageLocation?)
- addNewItemTapped -- show inline quick-add field
- newItemConfirmed -- create DetectedItemState from newItemName with 100 confidence, append, set isEditing
- bulkAddTapped -- add all checked items to pantry
- bulkAddCompleted(Result<Int, Error>)
- dismissTapped
- delegate(Delegate)

Delegate enum:
- itemsAdded(count: Int, itemNames: [String]) -- passes names for recipe matching
- dismissed

bulkAddTapped handler:
- Set isAdding = true, errorMessage = nil
- Get checked items
- Determine source: scanType == .fridge ? .fridgeScan : .receiptScan
- Call pantryClient.bulkAddScannedItems with mapped PantryItemInput array
  - Map estimatedExpiryDays to Date (Calendar.current.date(byAdding: .day, value: days, to: Date()))
  - Set userId from state
- On success: trigger UINotificationFeedbackGenerator.success() haptic on MainActor
- Send bulkAddCompleted(.success(count))

bulkAddCompleted(.success(count)):
- isAdding = false, addSuccess = true, addedCount = count
- Send delegate(.itemsAdded(count, itemNames))

bulkAddCompleted(.failure(error)):
- isAdding = false, errorMessage = error.localizedDescription
