**3. Extend ScanUploadReducer to call analyzeScan after upload:**

The reducer already has `@Dependency(\.apolloClient) var apolloClient` (see existing ScanUploadReducer pattern). Follow the exact same `.run` + `apolloClient.perform(mutation:)` pattern used in `compressionCompleted` for the upload mutation.

Add new actions:
- analysisCompleted([DetectedItem]) -- AI analysis returned results
- analysisFailed(String) -- AI analysis failed

Modify uploadCompleted handler:
- After setting state to .processing, return a `.run` effect that calls the analysis mutation:

```swift
case let .uploadCompleted(job):
    state.scanJob = job
    state.uploadState = .processing
    state.uploadProgress = 1.0
    let jobId = job.id
    let userId = state.userId
    return .run { send in
        await send(.delegate(.uploadStarted(job)))
        do {
            // 30-second timeout per locked user decision
            let result = try await withThrowingTaskGroup(of: AnalyzeScanMutation.Data?.self) { group in
                group.addTask {
                    let result = try await apolloClient.perform(mutation: AnalyzeScanMutation(jobId: jobId, userId: userId))
                    return result.data
                }
                group.addTask {
                    try await clock.sleep(for: .seconds(30))
                    throw CancellationError()
                }
                let first = try await group.next()
                group.cancelAll()
                return first
            }

            // Map GraphQL response to DetectedItem array
            guard let data = result, let response = data.analyzeScan else {
                await send(.analysisFailed("No response from analysis"))
                return
            }
            let items: [DetectedItem] = response.items.map { item in
                DetectedItem(
                    name: item.name,
                    quantity: item.quantity,
                    category: FoodCategory(rawValue: item.category) ?? .produce,
                    storageLocation: StorageLocation(rawValue: item.storageLocation) ?? .fridge,
                    estimatedExpiryDays: item.estimatedExpiryDays,
                    confidence: item.confidence
                )
            }
            await send(.analysisCompleted(items))
        } catch is CancellationError {
            await send(.analysisFailed("Analysis timed out. Please retry."))
        } catch {
            await send(.analysisFailed(error.localizedDescription))
        }
    }
```

Error handling:
- `CancellationError` from timeout -> `.analysisFailed("Analysis timed out. Please retry.")`
- Apollo network errors -> `.analysisFailed(error.localizedDescription)`
- Missing/nil response data -> `.analysisFailed("No response from analysis")`
- All failures surface in ScanUploadView with retry button

Add analysisCompleted handler:
- state.uploadState = .completed
- Send delegate(.analysisCompleted(items, scanJob)) -- add new delegate case

Add analysisFailed handler:
- state.uploadState = .failed
