- `deleteItem(UUID)` — Change to show confirmation alert first instead of deleting immediately. Set `state.itemToDelete` to the item, present AlertState with "Delete {item.name}?" title, destructive "Delete" button, and "Cancel" button.

**Alert handling:**
- On `.alert(.presented(.confirmDelete))`: Execute the actual delete (remove from items, call pantryClient.deleteItem). Clear itemToDelete.

**Delegate handling from AddEditItemReducer:**
- `.addEditForm(.presented(.delegate(.itemSaved)))`: Dismiss sheet (set addEditForm = nil in add mode; for batch add, the child reducer handles clearing internally — sheet stays open. Only dismiss on .itemSaved which is sent in edit mode). Re-fetch items with `.send(.onAppear)`.
  - Actually per CONTEXT: in add mode, the form stays open for batch add. The child reducer sends `.itemAdded` internally and only sends `.delegate(.cancelled)` when user taps Done. So: on `.delegate(.itemSaved)` (edit mode save), dismiss sheet and re-fetch. On `.delegate(.cancelled)`, just dismiss sheet and re-fetch.
- `.addEditForm(.presented(.delegate(.itemDeleted(let id))))`: Dismiss sheet, remove item from state.items, re-fetch.
- `.addEditForm(.presented(.delegate(.cancelled)))`: Dismiss sheet (set addEditForm = nil), re-fetch items to pick up any batch-added items.

**Enhanced PantryItemRow:**
Update the existing `PantryItemRow` to show:
1. **Expiry badge:** After the name, if item has expiryDate:
   - If expired (expiryDate < now): small red pill badge "Expired"
   - If expiring within 3 days (expiryDate < 3 days from now): small orange pill badge "Exp. soon"
   - Otherwise: no badge
2. **Subtitle line:** Below the quantity/category row, add optional line showing:
   - Expiry date formatted as "Exp: Mar 15" (if set, medium date style)
   - First line of notes truncated (if notes exist), separated by " · "
3. Keep the existing icloud.slash sync indicator on the trailing side

```swift
private struct PantryItemRow: View {
    let item: PantryItemState

    var body: some View {
        HStack {
            VStack(alignment: .leading, spacing: 4) {
                HStack(spacing: 6) {
                    Text(item.name)
                        .font(.body)
                    if let badge = expiryBadge {
                        Text(badge.text)
                            .font(.caption2)
                            .fontWeight(.medium)
                            .padding(.horizontal, 6)
                            .padding(.vertical, 2)
                            .background(badge.color.opacity(0.15), in: Capsule())
                            .foregroundStyle(badge.color)
                    }
                }

                HStack(spacing: 4) {
                    Text(item.quantity)
                    if let unit = item.unit { Text(unit) }
                    if let category = item.foodCategory {
                        Text("·"); Text(category.displayName).foregroundStyle(.secondary)
                    }
                }
                .font(.caption)
                .foregroundStyle(.secondary)

                // Subtitle: expiry date and/or notes
                if let subtitle = subtitleText {
                    Text(subtitle)
                        .font(.caption2)
                        .foregroundStyle(.tertiary)
                        .lineLimit(1)
                }
            }
            Spacer()
            if !item.isSynced {
                Image(systemName: "icloud.slash")
                    .font(.caption)
                    .foregroundStyle(.tertiary)
                    .accessibilityLabel(String(localized: "accessibility.pantry.not_synced", bundle: .main))
            }
        }
        .accessibilityElement(children: .combine)
    }

    private var expiryBadge: (text: String, color: Color)? {
        guard let expiry = item.expiryDate else { return nil }
        let now = Date()
        if expiry < now { return (String(localized: "pantry.badge.expired", bundle: .main), .red) }
        let threeDays = Calendar.current.date(byAdding: .day, value: 3, to: now)!
        if expiry < threeDays { return (String(localized: "pantry.badge.expiring_soon", bundle: .main), .orange) }
        return nil
    }

    private var subtitleText: String? {
        var parts: [String] = []
        if let expiry = item.expiryDate {
            parts.append("Exp: \(expiry.formatted(.dateTime.month(.abbreviated).day()))")
        }
        if let notes = item.notes, !notes.isEmpty {
            let firstLine = notes.components(separatedBy: .newlines).first ?? notes
            parts.append(firstLine)
        }
        return parts.isEmpty ? nil : parts.joined(separator: " · ")
    }
}
```
