   **Screenshot 1: Voice Narration** (01-voice-narration.png)
   - Navigate to: Recipe detail view with voice narration playing
   - Show: Mini player at bottom with play controls, recipe content visible above
   - Demo data: Pick recipe with great hero photo, have voice profile active
   - Headline overlay text suggestion: "Cook with Your Loved One's Voice"
   - Capture: Cmd+S in Simulator

   **Screenshot 2: Recipe Feed** (02-recipe-feed.png)
   - Navigate to: Main feed tab showing recipe cards
   - Show: 2-3 recipe cards with hero images, location badge, match % badge if pantry has items
   - Demo data: Curate feed with diverse, appetizing recipes
   - Headline overlay text suggestion: "Trending Recipes Near You"
   - Capture: Cmd+S

   **Screenshot 3: Pantry Scan** (03-pantry-scan.png)
   - Navigate to: Pantry tab, camera scan view active
   - Show: Camera viewfinder with ingredient detection overlay (or camera permission screen if scan not active)
   - Demo data: Have actual fridge/pantry visible in frame if possible
   - Headline overlay text suggestion: "Scan Your Pantry with AI"
   - Capture: Cmd+S (Note: Camera doesn't work in Simulator — capture on physical device or use a mock/static screen)

   **Screenshot 4: Dietary Filters** (04-dietary-filters.png)
   - Navigate to: Feed with dietary filter chips visible (or onboarding dietary selection step)
   - Show: Dietary preference chips (Vegan, Gluten-Free, etc.) with some selected
   - Demo data: Select 2-3 dietary filters to show personalization
   - Headline overlay text suggestion: "Personalized to Your Diet"
   - Capture: Cmd+S

   **Screenshot 5: Recipe Detail** (05-recipe-detail.png)
   - Navigate to: Full recipe detail view with ingredients and steps visible
   - Show: Recipe hero image, ingredient list, step-by-step instructions
   - Demo data: Pick a recipe with clear ingredient list and appetizing photo
   - Headline overlay text suggestion: "Step-by-Step Guidance"
   - Capture: Cmd+S

3. **Design overlay section:**
   - Template: 1320x2868px canvas in Figma/Photoshop
   - Layout: Top ~600-700px for gradient background + headline text, remaining for screenshot
   - Gradient: Warm brand colors (suggest: #FF6B35 orange to #FF8A5C light orange, or #E85D3A to #FF7F56)
   - Headline text: 70-90pt bold sans-serif (SF Pro Display Bold), white color, centered
   - Screenshot placement: Slight drop shadow, 20px padding from edges, rounded corners (40px radius)
   - Export: PNG format, verify 1320x2868px, under 10MB per file

4. **Naming convention:**
   - English: `fastlane/screenshots/en-US/01-voice-narration.png` through `05-recipe-detail.png`

   - 05: "Adim Adim Rehberlik" (Step-by-Step Guidance)

6. **Upload command:**

**Create Kindred/docs/app-icon-brief.md** with icon design brief:
- Size: 1024x1024px PNG, no transparency, no rounded corners (iOS auto-rounds)
- Style: Warm, cooking-themed, home/family feel
- Color palette: Warm oranges (#FF6B35, #FF8A5C), reds (#E85D3A), cream/white accents
- Imagery suggestions: Stylized cooking pot/pan with heart element, kitchen utensil silhouette, or abstract warm shape suggesting home cooking
- Avoid: Photographs, text, complex details (gets small on home screen)
- Inspiration: Similar warm, inviting cooking app icons
- Output: Single 1024x1024px PNG file to be placed at `Kindred/Sources/Assets.xcassets/AppIcon.appiconset/`
- Budget reference: If using a design service, $200-500 on 99designs or Fiverr, 3-5 day turnaround
