      - `purchaseCompleted(SubscriptionStatus)` ‚Äî updates status, hides paywall
      - `purchaseFailed(String)` ‚Äî sets error
      - `restoreTapped` ‚Äî calls restorePurchases, re-checks entitlement
      - `restoreCompleted(SubscriptionStatus)` ‚Äî updates status
      - `showPaywall` / `dismissPaywall` ‚Äî toggles showPaywall
      - `transactionUpdated(SubscriptionStatus)` ‚Äî from Transaction.updates stream
      - `clearError`
    - Effects:
      - `onAppear`: parallel load products + check entitlement + start transaction updates stream
      - `subscribeTapped`: set isPurchasing=true, call purchase on first product, send purchaseCompleted or purchaseFailed
      - `restoreTapped`: set isRestoring=true, call restorePurchases, re-check entitlement, send restoreCompleted
