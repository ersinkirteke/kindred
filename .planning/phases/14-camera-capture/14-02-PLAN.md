       - Extension on UIImage:
         - `func calculateSharpness() -> Double?` — Laplacian variance blur detection:
           - Convert to CIImage, apply CIPhotoEffectMono for grayscale
           - Apply CIConvolution3X3 with Laplacian kernel [0,-1,0,-1,4,-1,0,-1,0]
           - Calculate variance from output. Use CIContext to create CGImage.
           - Sample center region (not full image) for performance
           - Threshold: variance < 100 = blurry (return false for isSharp)
           - Wrap in autoreleasepool to prevent memory spike
         - `func compressForUpload(maxDimension: CGFloat = 2048, quality: CGFloat = 0.8) -> Data?`:
           - Autoreleasepool wrapping (CRITICAL — see Pitfall 2 in RESEARCH.md)
           - If max(width, height) > maxDimension, scale down proportionally using UIGraphicsImageRenderer
           - Return jpegData(compressionQuality: quality)
