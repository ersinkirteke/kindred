  <name>Task 1: Perform WCAG AAA color contrast audit of all color pairings</name>
  <files></files>
  <action>
Per locked decision: "Trust existing Asset Catalog values (already declare WCAG AAA 7:1 ratios) + manual contrast audit of all color pairings."

Audit every color pairing used in the app by reading Colors.swift and checking hex values against WCAG AAA requirements:

**Color pairings to verify:**

1. **Body text on backgrounds:**
   - kindredTextPrimary (#1A1A1A) on kindredBackground (#FFFFFF) — must be 7:1+
   - kindredTextPrimary (#1A1A1A) on kindredPrimary (#FFF8F0) — must be 7:1+
   - kindredTextPrimary (#1A1A1A) on kindredCardSurface (#FFF8F0) — must be 7:1+

2. **Secondary text:**
   - kindredTextSecondary (#6B5E57) on kindredBackground (#FFFFFF) — must be 4.5:1+ (large text) or 7:1+ (body)
   - kindredTextSecondary (#6B5E57) on kindredCardSurface (#FFF8F0)

3. **Accent text:**
   - kindredAccent (#C0553A) on kindredBackground (#FFFFFF) — must be 7:1+
   - kindredAccent (#C0553A) on kindredCardSurface (#FFF8F0) — must be 7:1+

4. **Buttons:**
   - White text on kindredAccent (#C0553A) — must be 4.5:1+ (large text buttons)
   - kindredTextPrimary on kindredAccent — must check

5. **State colors:**
   - kindredError (#C0392B) on kindredBackground (#FFFFFF)
   - kindredSuccess (#27AE60) on kindredBackground (#FFFFFF)

6. **Dark mode pairings:**
   - kindredTextPrimary (#F5EDE6) on kindredBackground (#1C1410)
   - kindredTextSecondary (#A89B93) on kindredBackground (#1C1410)
   - kindredAccent (#C0553A) on kindredBackground (#1C1410)

7. **Offline banner:**
   - White text on orange background

8. **ViralBadge, ForYouBadge:**
   - Badge text on badge background colors

Calculate contrast ratios using the formula: `(L1 + 0.05) / (L2 + 0.05)` where L1 and L2 are relative luminances (L1 > L2).

Document results in a contrast audit section within the SUMMARY.

**If any pairing fails WCAG AAA (7:1 for body, 4.5:1 for large text 18pt+):**
- Log the failing pairing with current ratio
- Suggest adjusted color value to achieve compliance
- Note: Per locked decision, NO programmatic tests — this is a manual audit documented in SUMMARY
